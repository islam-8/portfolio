SENTINEL — C-UAS Platform

Active Project · RPi5 · ARM64
The Challenge

Commercial drones represent a critical security threat — existing C-UAS solutions cost upward of $50,000 and require dedicated hardware. The challenge: build a real-time detection, tracking, and neutralization system on a $100 SBC that performs under 40ms latency.

The Solution

A multi-layer AI pipeline: MOG2 background subtraction triggers YOLOv8-nano inference every 3rd frame (skip=2), feeding a CSRT tracker and Kalman filter for smooth trajectory prediction. PTZ motorized camera auto-slews to bearing. RTL-SDR monitors RF spectrum in parallel. All running natively on Raspberry Pi 5.
